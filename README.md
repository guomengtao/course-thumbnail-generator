# 课程缩略图生成器 | Course Thumbnail Generator

[English](#english) | [中文](#中文)

## 中文

### 项目简介
课程缩略图生成器是一个简单易用的在线工具，用于快速创建美观的课程封面图片。

### 功能特点
- 🎨 自定义主题颜色和文字颜色
- 📝 支持添加标题和副标题
- 🎯 多种背景样式选择：
  - 渐变背景
  - 几何图案
  - 点阵图案
  - 波浪图案
- 💾 一键下载生成的缩略图
- 🖼️ 实时预览效果
